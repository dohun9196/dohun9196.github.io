DONGWON CONSTRUCTION

PHP 그누보드를 이용하여 서브페이지를 구현하였습니다.

MAIN PAGE 메인페이지는 FullPage로 구성이 되어있으며
MainVisul은 이미지 색상이 밝기 때문에 흰색의 텍스트가 묻히지 않도록 검정색 그라데이션을 넣었습니다.

SUB PAGE 서브페이지는 "회사소개", "사업분야", "홍보센터", "고객센터" 페이지 입니다.

NOTICE BOARD 게시판은 "홍보센터", "고객센터" 부분입니다. 고객센터의 문의접수 페이지는 로그인시 작동이 됩니다.
